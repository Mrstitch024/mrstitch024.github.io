常用的Mac版IntelliJ IDEA快捷键：

  1. 查找和替换：

    • 查找：Command + F
    • 替换：Command + R
    • 在项目中查找：Command + Shift + F
    • 在项目中替换：Command + Shift + R

  2. 编辑：

    • 复制行：Command + D
    • 删除行：Command + Delete
    • 移动行：Option + Shift + Up/Down
    • 注释/取消注释：Command + /
    • 块注释/取消块注释：Command + Option + /

  3. 导航：

    • 跳转到类：Command + O
    • 跳转到文件：Command + Shift + O
    • 跳转到行：Command + G
    • 跳转到定义：Command + B 或 Command + Click
    • 显示类结构：Command + F12

  4. 代码生成：

    • 生成代码（例如，get/set方法，构造函数等）：Command + N
    • 重构代码：Control + T

  5. 运行和调试：

    • 运行：Control + R
    • 调试：Control + D
    • 设置/取消断点：Command + F8

  6. 其他：

    • 打开/关闭项目视图：Command + 1
    • 打开/关闭控制台：Command + 4
    • 打开设置：Command + ,

以上只是一部分常用的快捷键，IntelliJ IDEA还有许多其他的快捷键可以使用。你可以在IntelliJ IDEA的”Preferences” -> “Keymap”中查看和自定义快捷键。
